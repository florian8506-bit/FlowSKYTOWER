# FlowSKYTOWER — Starter stable

Première base officielle du projet Godot.

## Test
- Ouvrir `project.godot` dans Godot 4.4+.
- Lancer avec F6/F5.
- Touches : 1 Rose, 2 Boost, 3 Attaque, 4 Super, R Reset.

## Structure
- `scenes/` scènes Godot
- `scripts/` logique du jeu
- `assets/` images et ressources
- `audio/` sons et musiques
- `effects/` particules et shaders
- `ui/` interface

Cette version est volontairement simple : elle sert de fondation propre avant les modèles 3D, animations, sons et connexion TikTok.
